유클피디아

김진돌 안허윽 강안녕 여인삼 
< 소개 >
김진돌은 유클팟의 첫째이자 수장이다.

실명 김나영과 별명 진돗개가 합쳐졌다.
( 별명 진돗개를 친근하게 진돌이 라고 불렀다. )

-TMI
현재 여친인 김주비(오너이름)가 있으며, 2년 넘게 연애중이다.

성적은 중하위권이며 공부는 안하고 컴퓨터에 미쳐있다.
사실 게임에 미쳐있다.
( 아니 기능반 때문에 미칠 것 같다 )

7살때부터 자신이 여자를 좋아한걸 알아챈 뼈 레즈다.

초등학생 때는 잘생겨서 헤테로도 반하게 한다는 마성의 레즈로 불리었으며,
지금은 해외를 다녀오면서 살과 나이를 먹으면서 훅 갔다.

엄마가 필리핀 사람으로 혼혈이다.

다른 사람들이 사주 좀 그만보라고 할 정도로 사주나 관상, 점, 타로를 많이 본다.
미신을 잘 믿는 편으로 지금은 줄였다.

키가 173cm까지 클 것이라는 목표(희망)을 가지고 있다.

운동을 좋아하는 편이다.

가수는 트로이 시반 이라는 해외 가수를 좋아한다.
( 잘생겼고 노래도 잘하고 퀴어야 완벽해 )

여가시간에는 bl이나 gl을 찾아먹거나 나투로를 보거나 유튜브, 트위터를 한다.


< 유클팟을 만든 이유 >
유클팟은 본래 친구와 같이 살기 위하여 만든 약속이다.
사실상 그저 약속이였기에 '유클팟'이라는 이름도 없었다.

( 유클팟 이전에 유치원팟 줄여서 유팟 또는 윽팟 이라고 불리었다. )

-비하인드
이 약속의 이름은 김진돌이 지은게 아닌 허윽이와 유팟인들이 즉흥적으로 지은것이 굳어진것이다.
수장도 모르는 사이에 사람이 불어나서 한 단체를 이루게 되었다는..

( 원래는 허락을 받고 들어와야 하지만..지금은 귀찮아서 다 허락해줬다 )
